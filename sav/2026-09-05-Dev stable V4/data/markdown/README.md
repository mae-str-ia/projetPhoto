# Markdown texte

Ce dossier regroupe les fichiers Markdown utilises pour generer le PDF texte du livre.

- `source/` : fichiers Markdown importes tels quels depuis Word/Pandoc ou une autre source.
- `clean/` : fichiers Markdown nettoyes, normalises en UTF-8 et prets pour la generation PDF.
- `build/` : fichiers intermediaires generes automatiquement.

Le PDF final genere a partir du Markdown doit ensuite etre place dans `data/pdf/source.pdf`.
