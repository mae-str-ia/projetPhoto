# Typst Tester

Outil pour valider les styles Typst (titres, couleurs, mise en page, etc.).

## Utilisation

```bash
php generate-test.php
```

Compile `test-styles.typ` et génère les PDFs de test dans `outputs/`.

## Fichiers

- `test-styles.typ` — Template Typst avec différents styles
- `generate-test.php` — Script de génération
- `outputs/` — PDFs générés
