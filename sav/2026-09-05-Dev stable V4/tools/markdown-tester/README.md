# Markdown Tester

Outil pour valider le support des différents éléments markdown.

## Utilisation

```bash
php preview.php --file samples/test-headings.md
```

Affiche le rendu HTML du fichier markdown.

## Fichiers Exemples

- `samples/test-headings.md` — Titres et hiérarchie (H1, H2, H3, etc.)
- `samples/test-formatting.md` — Mise en forme (gras, italique, code, listes, etc.)
- `samples/test-special.md` — Éléments spéciaux (images, liens, tableaux, etc.)

## Scripts

- `preview.php` — Aperçu des samples markdown
