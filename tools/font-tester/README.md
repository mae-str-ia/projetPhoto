# Font Tester

Outil pour tester les différentes polices disponibles en PDF.

## Utilisation

```bash
php test-fonts.php
```

Génère des PDFs d'exemple dans `samples/` avec chaque police testée.

## Fichiers

- `test-fonts.php` — Script PHP de test
- `test-fonts.html` — Aperçu HTML des polices
- `samples/` — PDFs générés
